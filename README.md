# About

`netblackhole` 可以用来模拟基于TCP/IP协议的网络服务，但是它对于任何请求都不做回应。
这看起来没什么用，但你可以在单元测试中用来检验你的程序中对远程调用的容灾处理是否恰当。

# Install

```bash
$ npm install netblackhole
```
